# 🧿 Pardus Anadolu

**Anadolu’nun sıcaklığı, Pardus’un gücüyle buluşuyor.**

Pardus Anadolu, Türkiye’nin açık kaynak işletim sistemi Pardus için tasarlanmış özgün ikon ve duvar kâğıdı paketidir. Bu proje, Anadolu Linux ruhunu yaşatırken Pardus’un resmi ekosistemine kültürel bir katkı sunmayı amaçlar.

## 🎯 Amaç

- Pardus kullanıcılarına estetik ve kültürel bir masaüstü deneyimi sunmak
- Hamonikr ikonları, kilim motifleri, bağlama esintileriyle özgün tasarımlar üretmek
- Cinnamon + Plank masaüstü ortamı için özel temalar geliştirmek
- Açık kaynak topluluğuna katkı sağlamak ve genç kullanıcıları teşvik etmek

## 📁 İçerik

- `icons/` → Anadolu temalı ikon setleri (SVG/PNG)
- `wallpapers/` → Yüksek çözünürlüklü duvar kâğıtları (Aşık Veysel, Neşet Ertaş, sahil vb.)
- `themes/` → Cinnamon + Plank için GTK, Metacity, GRUB ve LightDM temaları
- `README.md` → Proje açıklaması ve katkı yönergeleri
